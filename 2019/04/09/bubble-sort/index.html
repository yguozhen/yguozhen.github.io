find patterns

Bubble sort is the pattern that from first number x = array[0] to calculate. If compared number array[1] is bigger than the x , then x = array[1] . The first circle is to find the biggest number and the second circle is to find the biggest number except for the biggest number you find before this circle.

check by hand

First circle array[] = {4,0,-1,-2,21}

second array[] = {0,-1,-2,4,21}
…….
.

translate to code

public class Demo4 {
public static void main(String[] args) {
    //circle number
    int []arr= {4,21,0,-1,-2};
    for(int i=0;i<arr.length-1;i++) {
        //comparative number
        for(int j=0;j<arr.length-i-1;j++) {

            if(arr[j] > arr[j+1]) {
                int temp = arr[j+1];
                arr[j+1] = arr[j];
                arr[j] = temp;
            }


        }
    }
    for(int k:arr){
        System.out.print(k+" ");
    }

}

}
